## 📌 About This Repository

This repository tracks my progress, projects, notes, and learnings from the **Codebasics Gen AI & Data Science Bootcamp 3.0**. The bootcamp is designed by industry hiring managers with 8+ years of international data industry experience, focused on making learners job-ready for roles like:

- 🧠 Data Scientist
- ⚙️ Machine Learning Engineer
- 🗣️ NLP Engineer
- 🤖 AI / Gen AI Engineer

---

## 🗺️ Bootcamp Curriculum Overview

### 🔷 Module 1 — Foundations

| Topic | Description |
|---|---|
| **Python** | Core programming, OOP, data structures, real-world problem-solving |
| **SQL** | Querying, joins, aggregations, window functions, business use cases |
| **Math & Statistics** | Probability, distributions, hypothesis testing, statistical inference |

---

### 🔷 Module 2 — Machine Learning

| Topic | Description |
|---|---|
| **ML Fundamentals** | Supervised & unsupervised learning, regression, classification, clustering |
| **Feature Engineering** | Data cleaning, encoding, scaling, selection techniques |
| **Model Evaluation** | Cross-validation, metrics (accuracy, F1, AUC-ROC), bias-variance tradeoff |
| **ML Project Lifecycle** | Data collection → Training → Evaluation → Deployment → Monitoring |

---

### 🔷 Module 3 — Deep Learning

| Topic | Description |
|---|---|
| **Neural Networks** | Perceptrons, activation functions, backpropagation |
| **CNNs** | Image classification and computer vision fundamentals |
| **RNNs / LSTMs** | Sequence modeling, time-series forecasting |
| **Transfer Learning** | Pre-trained models and fine-tuning strategies |

---

### 🔷 Module 4 — Natural Language Processing (NLP)

| Topic | Description |
|---|---|
| **Text Preprocessing** | Tokenization, stemming, lemmatization, stop words |
| **Classical NLP** | TF-IDF, Bag of Words, sentiment analysis |
| **Transformers & BERT** | Attention mechanisms, pre-trained language models |
| **NLP Pipelines** | End-to-end NLP project workflows |

---

### 🔷 Module 5 — Generative AI

| Topic | Description |
|---|---|
| **LLM Fundamentals** | How large language models work, capabilities and limitations |
| **Prompt Engineering** | Prompt design, few-shot learning, chain-of-thought |
| **RAG (Retrieval-Augmented Generation)** | Vector databases, embeddings, building RAG pipelines |
| **Hallucinations & Safety** | Model limitations, cost trade-offs, responsible AI usage |
| **Tools** | LangChain, LangGraph, vector stores (FAISS, Chroma) |

---

### 🔷 Module 6 — Agentic AI

| Topic | Description |
|---|---|
| **Agent Architecture** | ReAct agents, planning, memory, tool use |
| **Multi-Agent Systems** | Orchestrating multiple agents, CrewAI, LangGraph workflows |
| **MCP (Model Context Protocol)** | Agent communication protocols, context management |
| **Observability & Evaluation** | LangSmith, cost tracking, debugging agent pipelines |
| **Tools** | CrewAI, Amazon Bedrock AgentCore, LangGraph |

---

### 🔷 Module 7 — Deployment & Production

| Topic | Description |
|---|---|
| **API Development** | Building REST APIs with FastAPI |
| **Frontend for AI Apps** | Streamlit dashboards and interactive UIs |
| **Cloud Deployment** | Deploying models and apps on cloud platforms |
| **MLOps Basics** | CI/CD for ML, model monitoring, version control |

---

### 🔷 Module 8 — Virtual Internship

- Simulated real-client environments with ambiguous requirements and imperfect data
- Deliverables: data cleaning, validation, deployment, and debugging workflows
- Collaborative workflow experience mimicking real industry teams
- Internship certificate upon completion
